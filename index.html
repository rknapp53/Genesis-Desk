📖 GENESIS·DESK — USER GUIDE

1 · Getting Data In

This tool reads ATCF a-deck files — plain-text tropical cyclone forecast data. There are two independent upload slots on the home screen, and you can fill in either one or both at once:

Source 1 — Standard ATCF (RAL / NHC)

  • Go to hurricanes.ral.ucar.edu/realtime/current and click on the storm you want.
  • Scroll to the bottom of that storm's page to the list of ATCF links.
  • The a-deck file is always the first link in that ATCF link list — click it to download.
  • Upload that file into Source 1 on the home screen.

Source 2 — Google WeatherLab

  • Go to deepmind.google.com/science/weatherlab and hit the Download Data button in the top right corner, next to the settings gear icon.
  • Select our experimental model, then Ensemble, then ATCF.
  • This is also where you pick which run to use — choose the most recent DeepMind run available.
  • Hit the download link to get the file, then upload it into Source 2 on the home screen.

Both accept .dat, .dat.gz (gzip is decompressed automatically), or plain .txt. Once at least one file is chosen, the Load Data button lights up — click it to parse and open the storm view.

If you upload both sources for the same storm, everything from each stays independently selectable — nothing gets blended together.

2 · If Your File Has Multiple Storms

Some ensemble runs cover more than one active storm at once. If the app detects this, an orange "⚠ N storms detected in this file" dropdown appears at the top of the side panel. Pick a storm and the map automatically recenters and every chart/metric updates for that storm. It defaults to whichever storm has the most data in the file.

3 · The Map Toolbar

Two rows sit above the map:

  • DATA row — toggle which track groups are drawn: Ensembles, Track Models, Intensity Models, and Google Ensemble. Any group with no data in your file(s) greys out automatically. MSLP Heatmap paints a soft pressure-based overlay from whichever ensemble source(s) are on. Cities toggles reference city labels (more appear as you zoom in). Wind Units (KT/MPH) affects the whole app at once. Color Map Tracks By switches track lines between a flat per-source color, or a Wind Speed / Pressure gradient — this control greys out while the heatmap is active, since the two overlap.
  • ANALYSIS row — scoped to whichever tab is selected in the side panel (Ensembles / Intensity Models / Google). Uncertainty Cone shades the spread of that group's tracks out to +84h. Strike Probability shows a grid of what % of members pass through each map cell. The Play control (▶, scrubber, time readout, ✕ reset) animates the forecast forward in time — drag the scrubber and the map updates live while the side panel syncs once you let go, so it stays smooth even on long forecasts.

4 · Clicking the Map

Click anywhere on the map to drop a pink target marker with a dashed radius circle. This feeds the Closest Approach calculator in the side panel — no need to type coordinates unless you want precision, though that option's there too.

5 · The Side Panel

  • Cycle selector — switch between forecast runs/cycles present in the file.
  • Ensembles / Intensity Models / Google tabs — choose which model group every metric, chart, and analysis tool below is based on. Greys out if that source isn't in your data.
  • Metrics — current wind/pressure, and the mean peak wind / mean minimum pressure across the selected group.
  • Rapid Intensification flag — warns if any member shows a ≥30kt wind jump within a 24h window, with the earliest onset time and which members triggered it.
  • Model Agreement — an overall 0–100 score plus a line chart showing how tightly clustered the guidance is across the whole forecast (useful for spotting "confident early, diverging later" patterns). Hover the line for exact numbers; NEW TAB pops out a larger standalone version.
  • Closest Approach — after clicking the map (or entering coordinates), shows what % of members pass within your chosen radius (in miles) and the earliest / median arrival time.
  • Probability Distribution — % of members reaching TD / TS / Cat 1–5 wind thresholds.
  • Wind & Pressure charts — full spaghetti of every member over time, with real UTC time on the x-axis. Hover any line for a tooltip; NEW TAB opens a bigger, fully live version with its own KT/MPH toggle.

6 · Good to Know

  • Click the GENESIS·DESK logo (or the ↺ New File button in the side panel) anytime to return to the upload screen.
  • Everything runs entirely in your browser — files are never uploaded to a server.
  • This guide is always one click away via the 📖 User Guide button, both here and inside the main view.
